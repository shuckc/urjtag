About

UrJTAG aims to create an enhanced, modern tool for communicating over JTAG with flash chips, CPUs, and many more. It takes on the well proven openwince jtag tools code. Future plans include conversion of the code base into a library that can be used with other applications. A flexible remote communication protocol that can be used over almost any type of serial link (including TCP/IP) is currently being defined.

Status

Since release 0.7, support for reading BSDL descriptions natively has been added, USB cable transfers are much faster, and new bus and cable drivers have been added as well as numerous other improvements and bug fixes. The internal structure was straightened, to ease conversion of the existing code base into a library in near future.

Version 0.8 has been released on February, 20th and is now available in the Downloads section!
